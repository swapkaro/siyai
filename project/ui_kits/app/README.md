# UI Kit · SiyAI App (Studio + Library)

The flagship surface. A storyteller arrives, writes (or speaks) a memory, chooses a format (film / song / letter), and watches it come to life.

- `index.html`. Studio + Library. Sidebar with collections, hero prompt with mode chooser, library grid (with one piece in-progress).

- **Frosted topbar** (sticky, glass)

- **Mode segmented control** (Film / Song / Letter)
- **Suggestion chips**. Italic emotional phrases

- **Glass play FAB** on each poster
- **Mono metadata strip** under each title
