# UI Kit · SiyAI Mobile

Two phone screens demonstrating that *"mobile feels as magical as desktop."*

## Screens

- **Home**. Greeting, prompt, recent stories carousel, dark floating bottom nav (pill-shaped, frosted).
- **Now Playing**. Full warm gradient background with breathing orb. The orb IS the artwork.

## Patterns

- **Floating glass bottom nav** in dark ink. Sits *over* content, never in a flat bottom bar.
- **The orb** is the brand's signature artwork. It appears on home, processing states, and now-playing.
- **Hero prompt on mobile** mirrors desktop. Same italic serif placeholder, same warmth.
- All surfaces stay 16–24px from screen edges. Cinematic margin even on phone.
