# UI Kit · SiyAI Marketing

Cinematic landing experience. The marketing site is *quiet*, *editorial*, and *literary*. A publication, not a SaaS page.

## Sections

- `index.html`. Full landing page: hero, showcase row, pull quote, how-it-works, atmospheric strip, dark CTA, footer.

## Beats

1. **Hero**. Huge italic serif headline ("Where stories come alive."), warm orb, single primary CTA + watch-a-piece link.
2. **Trust strip**. Featured publications. No client logos. A publication list feels more literary.
3. **Showcase**. Three cinematic posters with italic titles, asymmetric grid.
4. **Pull quote**. Full-bleed dawn-wash background, 88px italic serif quote.
5. **How it works**. Three numbered steps, editorial column layout.
6. **Atmospheric strip**. Pure mood. 320px of warm gradient + italic line.
7. **Dark CTA**. Inverted moment ("What will you make today?") with gold italic accent.
8. **Footer**. Dark, minimal.
